# Changelog

## 0.5.2 - Khaos-Krew Organization Release

- Moved the official update and download home to `Khaos-Krew/nexus-idleon-companion`.
- Updated the built-in stable release feed to the Khaos-Krew organization.
- Updated source module and documentation references for the organization repository.
- Preserved the existing app-data folder for seamless upgrades.

## 0.5.1 - Nexus Idleon Companion

- Renamed Idleon Account Monitor to Nexus Idleon Companion.
- Added a beginner-oriented product identity and clearer release documentation.
- Prepared the application for public GitHub distribution.
- Added the default GitHub update-feed location.

## 0.5.0 - Guided Progress Update

- Added a first-run guided tour and upgrade What's New screen.
- Added beginner-friendly Guided Mode with a smaller navigation set.
- Added plain-language explanations across the command center.
- Added a dedicated Character Progress page with per-character progression scores.
- Added setup meters for core equipment, tools, food, cards and anvil production.
- Added strengths, watchouts, priority steps and role guidance for each character.
- Added character level history and one-click recommendation-to-goal conversion.
- Added the Update Center with HTTPS release-feed checking, semantic version comparison, channel selection, SHA-256 validation and staged Windows replacement.
- Added manual executable update selection for offline or unpublished releases.
- Added local backup creation and validated backup restoration.
- Added privacy-safe support diagnostics.
- Added configurable starting page and guided/advanced navigation preferences.
- Added explicit integration schema versions to health and latest-account responses.

## 0.4.0 - Command Center

- Added Today dashboard, goals, timers, rules, roles, system planners, analytics and snapshot comparison.
- Added a privacy-controlled Khaos Nexus integration export.
- Added full local workspace persistence and inventory aggregation.

## 0.3.0 - Review and Visuals

- Added account review scoring, prioritized findings and visual character loadouts.
- Added cached item, card, class, talent and account-system icons.

## 0.2.0 - Native Windows Build

- Replaced the Node/Electron requirement with a native standalone Windows executable.

## 0.1.0 - Initial MVP

- Added read-only Idleon LevelDB capture, JSON import, snapshots and basic recommendations.
